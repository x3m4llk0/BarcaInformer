# BarcaInformer Roadmap

Этот файл - главный трекер проекта. Идем по эпикам сверху вниз и отмечаем выполнение по мере готовности.

## Epic 1. Football Provider Foundation

Цель:
- собрать минимальный источник данных по матчам Барселоны через `football-data.org`

Результат:
- [x] сервис умеет получать ближайший матч
- [x] внешняя DTO отделена от внутренней модели `MatchInfo` на базовом уровне
- [x] `main` может вызвать provider и получить осмысленный результат
- [ ] provider cleanup и полная нормализация ошибок перенесены в рефакторинг

Темы:
- `net/http`
- `encoding/json`
- маппинг DTO -> domain model
- обработка ошибок

## Epic 2. Telegram Bot Command Flow

Цель:
- подключить Telegram-бота как входную точку взаимодействия

Результат:
- [x] подключена библиотека `go-telegram-bot-api/telegram-bot-api/v5`
- [x] бот работает через long polling (`getUpdates`)
- [x] реализована команда `/start`
- [x] реализована команда `/match`
- [x] по `/match` бот получает матч из provider и отвечает в `chat.id` из входящего апдейта
- [x] ручная отправка в захардкоженный чат заменена на command flow
- [x] добавлена базовая защита от `nil`-update
- [x] логика команд отделена от логики получения футбольных данных на базовом уровне
- [ ] cleanup ошибок Telegram-части и финальная полировка handler/notifier перенесены в рефакторинг

Темы:
- Bot API
- `go-telegram-bot-api/v5`
- polling
- update/message/command handling
- формирование ответа пользователю
